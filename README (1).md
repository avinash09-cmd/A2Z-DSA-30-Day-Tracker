# A2Z DSA — 30 Day Tracker

A single-file, self-contained HTML checklist for working through Striver's A2Z DSA Sheet in 30 days — **Trees, Graphs, and Dynamic Programming excluded** to keep the timeline realistic.

## What's inside

- **30 days**, grouped into 4 weekly blocks:
  - Week 1 — Foundations + Arrays
  - Week 2 — Binary Search + Strings
  - Week 3 — LinkedList + Recursion + Bit Manipulation
  - Week 4 — Stack/Queue + Heaps + Greedy + Final Revision
- Each day lists the topic and the specific concepts/problems to cover.
- Days tagged **MUST** are the highest-priority topics — do these first if you fall behind schedule.
- Built-in revision days (7, 14, 21, 28, 29, 30) so you're not just solving once and forgetting.

## How to use it

1. Open `a2z_dsa_30day_tracker.html` in any web browser (double-click the file, or open it via **File → Open** in your browser).
2. Check off each day as you complete it — the progress bar at the top updates automatically.
3. Your progress is saved automatically and will still be there next time you open the file.
4. Fell behind? Use the **MUST**-tagged days as your minimum viable path — everything else is a bonus if time allows.
5. Use **"reset all progress"** at the bottom if you want to start over.

## Notes

- This is a planning/tracking tool, not the problem sheet itself — pair it with Striver's actual A2Z DSA Sheet (search "Striver A2Z DSA Sheet takeUforward") for the problem links.
- No internet connection or account needed — everything runs locally in your browser.
- If you're prepping specifically for TCS NQT rather than general interviews, note that the coding round there rarely goes beyond arrays, strings, and basic math/recursion — so Weeks 1–3 matter most for that exam specifically.
